# be-ui3
基于vue3.0的简单ui组件库
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

See [Configuration Reference](https://cli.vuejs.org/config/).
